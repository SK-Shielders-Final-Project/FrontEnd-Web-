현재 2단계 인증(2FA) 로그인 로직을 구현 중입니다.

[요구 사항 1: 1차 로그인 성공 처리] 1차 로그인 API 응답으로 아래와 같은 JSON을 받습니다:

JSON
{
    "is2faEnabled": false,
    "tempAccessToken": "...",
    "requires2FA": true,
    "success": false,
    "tempRefreshToken": "...",
    "userId": 22
}
이 응답을 받으면 다음 매핑으로 LocalStorage에 임시 저장하는 로직을 작성해 주세요.

tempAccessToken -> 키 이름: "token"

tempRefreshToken -> 키 이름: "refreshToken"

userId -> 키 이름: "adminId"

[요구 사항 2: OTP 인증 성공 처리] otpVerificationPage에서 인증번호 검증이 성공했을 때 실행될 로직입니다.

LocalStorage에 저장해 둔 "token", "refreshToken", "adminId" 값을 가져옵니다.

가져온 값을 setCookie 함수를 사용하여 쿠키에 저장합니다.

중요: 쿠키에 저장할 때 사용하는 키(Key) 이름은 현재 프로젝트에서 어드민 토큰 관리에 사용하는 상수/키 값(예: ACCESS_TOKEN_KEY 등)을 그대로 사용해서 세팅해 주세요. (만약 정확한 키 상수를 모른다면 주석으로 표시해 주세요).

쿠키 세팅 후 LocalStorage의 임시 값들은 삭제(removeItem)해 주세요.
