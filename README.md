# ZDME-Front

ZDME 프론트엔드 애플리케이션

[Create React App](https://github.com/facebook/create-react-app)을 사용하여 구축


## 🚀 시작하기

이 가이드는 로컬 개발 환경에서 프로젝트를 설정하고 실행하는 방법을 안내합니다.

---

### ✅ 1. [Node.js](https://nodejs.org/)**v22** 버전 설치

**Windows 권장 설치 방법 (`nvm-windows`)**

1. [Node.js 공식 홈페이지](https://nodejs.org/en/download/releases)에서  `*.msi` 설치 파일을 받아 설치 파일을 다운로드하고 실행
2. 관리자 권한으로 **PowerShell** 또는 **CMD**를 열고 다음 명령어를 입력하여 Node.js 버전 확인

   ```shell
   npm -v
   ```

> 또는, `nvm-windows` 설치 후 node.js 버전 관리 가능

---

### 💻 2. 프로젝트 설치

필요한 패키지를 설치합니다.

```powershell
npm install
```



---

### ▶️ 3. 개발 서버 실행

아래 명령어를 실행하여 개발 서버를 시작합니다.

```bash
npm start
```

서버가 성공적으로 시작되면 자동으로 브라우저가 열리고 [http://localhost:3000](http://localhost:3000) 주소로 애플리케이션이 실행
